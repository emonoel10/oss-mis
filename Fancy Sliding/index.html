ABOUT


          This page is designed to help school personnel, especially DNSC OSS-guidance Personnel, create a web-based Socio-economic profiling & Graduate Tracer Survey. When a school is conducting enrolment or collecting data via online.
          The user code requires that DNSC OSS-Guidance personnel have some sort of privacy policy on this website. To meet user code, privacy policies must be based on the following four elements:

DNSC Graduates

  • General Information
  • Educational Background
  • Training Courses
  • Employment Status
  • Confirm
